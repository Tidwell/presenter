How does this effect day-to-day coding?

New CMS forms should be written using knockout

New CMS components should be written as jQuery plugins w/ knockout adapters

Library code should be namespaced appropriately in the BI hierarchy

No more generating CMS forms with PHP

All non-library JS components should be encapsulated into modules
